## Print-Friendly Checklist

### Basic Diagnostic Flow
```
[ ] 1. Read codes with scanner
[ ] 2. Document ALL codes (write them down)
[ ] 3. Google code + "1997 Impreza" or check this guide
[ ] 4. Start with easiest fix (gas cap, air filter)
[ ] 5. Test sensors with multimeter (see specs above)
[ ] 6. Fix root cause (don't just clear codes)
[ ] 7. Clear codes, drive 50+ miles, recheck
```

### Pre-Purchase Inspection
```
[ ] Scan for codes (current + pending + history)
[ ] Check oil level and condition (dark brown OK, black = overdue)
[ ] Coolant level and color (green/orange, not rusty)
[ ] Transmission fluid (red, not burnt smell)
[ ] Exhaust smoke (white = head gasket, blue = oil burning)
[ ] Test drive: smooth idle, no hesitation, normal temp
[ ] Underbody: oil leaks (valve covers common), rust
```
